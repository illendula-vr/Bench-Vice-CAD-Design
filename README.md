## Overview
This project presents the complete design and development of a Bench Vice using CAD software. The model was created to demonstrate mechanical design principles, assembly modeling, engineering drawing generation, and design-for-manufacturing considerations.

## Project Objectives
- Design a functional bench vice assembly.
- Develop detailed 3D CAD models of individual components.
- Create a complete assembly with proper constraints.
- Generate manufacturing drawings for fabrication.
- Improve practical mechanical design and drafting skills.

## Design Features
- Fixed and movable jaw arrangement
- Lead screw driven clamping mechanism
- Ergonomic handle design
- Stable base structure
- Fully assembled and constrained CAD model
- Manufacturing-ready engineering drawings
